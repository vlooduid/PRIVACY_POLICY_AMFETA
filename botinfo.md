# Discord Bots Overview

## Utility Bot

Features:

* Role management
* AI assistant
* Event announcements
* News system
* Translation tools
* User interaction commands
* Support tickets
* Staff applications
* Report system
* Utility commands

## Economy Bot

Features:

* User profiles
* Economy system
* Virtual inventory
* Cases and rewards
* Reputation system
* Marriage system
* Activity tracking
* Mini-games
* Social interactions

Stored data:

* User IDs
* Economy balances
* Inventory items
* Activity statistics
* Economy progress

## Moderation Bot

Features:

* Warn
* Mute
* Kick
* Ban
* Clear
* Moderation history
* Invite tracking

Stored data:

* User IDs
* Moderation records
* Invite tracking information

## Security Bot

Features:

* Raid protection
* Anti-spam
* Anti-advertising
* AI-assisted moderation
* User verification

Stored data:

* User IDs
* Moderation records
* Security logs

## Information Bot

Features:

* Help system
* Server banner management
* Voice channel presence

The bot currently stores no user data.
